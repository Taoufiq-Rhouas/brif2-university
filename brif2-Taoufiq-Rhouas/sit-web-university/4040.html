application de suivie des élèves.



Plusieurs utilisateurs peuvent noter des incidents ou bonus sur une unique base de données. Cela peut permettre à plusieurs enseignants de faire un suivi en équipe et en temps réel.




1. Se connecter à la console Firebase, avec une adresse gmail, pour créer une base de données partagées.

https://firebase.google.com/


2. Cliquer sur 'Accéder à la console'


3. Ajouter un projet, donner un nom au projet et cliquer sur 'créer un projet'


4. A gauche, cliquer sur 'Database'. Puis cliquer sur l'onglet 'Règles' et coller le code suivant pour autoriser l'écriture et la lecture aux utilisateurs. Puis cliquer sur 'Publier'


{

"rules":{

".read": true,

".write": true

}

}


5.Cliquer ensuite sur 'Settings', 'Paramètres du projet' et' Ajouter Firebase à votre application Android'


6.Coller la chaine com.ga.suivi4

https://firebase.google.com/


7.Cliquer sur 'Enregistrer l'application' , 'Continuer' et 'Terminer'


8.Mémoriser l'adresse (ex. : https://xxxxxx-xxxxxxx.firebaseio.com/) de la base de données.Elle sera saisie dans l'application.

https://www.youtube.com/watch?v=1yGsPTM9Fxw
